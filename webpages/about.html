What brought us together!

We met and clicked in the Code First Girls web design workshops taught at LSE. Code First Girls is a project aiming to increase the number of women in tech. Learn more about it and get involved here .
The idea behind this project was to create a food travel guide tool that gives you a grasp of different world tastes – all compiled in one easily accessible platform. We hope that our initiative will inspire you to travel and learn more about various cuisines around the world!
